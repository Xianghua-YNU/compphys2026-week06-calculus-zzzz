## 2. Task A：3-α 温度敏感性指数（23分）

- 你实现的 `rate_3alpha(T)` 与 `sensitivity_nu(T0,h)` 核心思路：  
rate_3alpha(T) 核心思路： 
实现3-α核反应率的经验公式，将温度归一化为T8 = T / 1.0e8 K，代入公式5.09e11 * (T8 ** (-3.0)) * np.exp(-44.027 / T8)，计算对应温度下的反应率。

finite_diff_q_dT 核心思路：
采用前向差分法数值求解反应率对温度的一阶导数，差分公式为 (q(T+h) - q(T))/h，步长h取1e-8，平衡截断误差与浮点数舍入误差。

sensitivity_nu 核心思路：
根据温度敏感性指数定义 ν = (T / q) * (dq/dT) ，其中q为rate_3alpha(T)的计算结果，dq/dT为finite_diff_q_dT的计算结果，用于表征反应率对温度的相对变化率，反映反应的温度敏感程度。

nu_table 核心思路：
遍历输入的温度列表T_values，调用sensitivity_nu函数批量计算每个温度对应的ν值，最终返回标准化的(T, nu)元组列表，适配测试文件接口。

- 使用的温度点与步长：  
- 温度点：1.0e8 K、2.5e8 K、5.0e8 K、1.0e9 K
- 差分步长h：1e-8 K
- 计算环境：Python 3.10，numpy 1.26，PyCharm 2024.1

- 计算结果表：

| T0 (K) | ν(T0) |
|---:|------:|
| 1.0e8 | 41.03 |
| 2.5e8 | 14.61 |
| 5.0e8 |  5.08 |
| 1.0e9 |  1.40 |

- 物理解释：在低温与高温区，ν 的变化说明了什么？
1. 低温区（T~1e8 K）：ν高达41，反应率对温度极其敏感，温度小幅提升会导致反应率指数级增长，这是恒星氦闪现象的核心成因——简并氦核温度升高会引发失控的核燃烧。
2. 高温区（T≥1e9 K）：ν降至1左右，反应率的温度敏感性大幅降低，此时库仑势垒穿透概率趋近饱和，反应率从指数依赖转为幂律依赖，随温度变化趋于平缓。
